How to use

First thing you need to do is download the vertion of the game you want to play. next create a folder called "Dungeon Master Files". next drag the .bat file into that folder. next go to computer or file explorer on windows 10, then find the drive you want to save the game to, then find ProgramFiles (x86) and drag the folder you just made into there. lastly you will create a shortcut on your desktop and link it to the .bat file. and lastly name the file and change the icon.
